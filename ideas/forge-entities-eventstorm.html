Проблемы / Hotspots

Marketplace API

Discover ищет только локально. Удалённый поиск не работает.

Skill Chaining

Вручную вызывать brainstorm → plan → execute. Нет автоцепочки.

Learning Extraction

Не извлекает инсайты из ошибок и коммитов автоматически.

Cross-Project

Нет переноса learnings между проектами. Каждый .forge/ изолирован.

Auto-Detection

Только keyword matching. Нет семантического понимания.
