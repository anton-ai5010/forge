# Звуки

Голосовые уведомления для плагина — воспроизводятся автоматически когда Клод заканчивает ответ или просит разрешение что-то сделать. Каждый раз выбирается случайный файл из нужной папки.

## permission/

Голоса для запроса разрешения — звучат когда Клод хочет выполнить действие и ждёт подтверждения от пользователя.

- **razreshenie1.mp3** — голосовое уведомление о запросе разрешения (вариант 1)
- **razreshenie2.mp3** — голосовое уведомление о запросе разрешения (вариант 2)
- **razreshenie3.mp3** — голосовое уведомление о запросе разрешения (вариант 3)
- **razreshenie4.mp3** — голосовое уведомление о запросе разрешения (вариант 4)
- **razreshenie5.mp3** — голосовое уведомление о запросе разрешения (вариант 5)
- **razreshenie6.mp3** — голосовое уведомление о запросе разрешения (вариант 6)
- **razreshenie7.mp3** — голосовое уведомление о запросе разрешения (вариант 7)

## stop/

Голоса завершения — звучат когда Клод заканчивает отвечать, чтобы не надо было смотреть на экран в ожидании.

- **ElevenLabs_..._Roger_....mp3** — голос Roger (расслабленный, разговорный) — уведомление о завершении
- **ElevenLabs_..._VASKO_...mp3** (6 файлов) — голос VASKO (глубокий, тёплый) — варианты уведомления о завершении с разными параметрами синтеза
